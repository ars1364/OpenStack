# Custom kolla configs (overlays applied on top of upstream)

Octavia client/server CA cert + key live here when the LBaaS feature is on, generated by kolla-ansible octavia-certificates. Not committed - they are unique per deploy. Run `kolla-ansible octavia-certificates` after the first deploy, then chmod 0644 the .pem files (kolla-ansible 2026.1 reads them as the connecting ubuntu user, not root).
